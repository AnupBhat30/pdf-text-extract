# Document Unroller

A premium, secure, and fast document text extractor. Extract text from PDF, Word, and text files instantly in your browser.

## Features

- **Premium UI:** Apple-inspired minimalist design with dark mode support.
- **Privacy First:** All processing happens locally in your browser. Your documents never leave your computer.
- **Wide Support:** Works with PDF, DOCX, TXT, MD, RTF, CSV, JSON, XML, HTML, and LOG files.
- **Batch Processing:** Upload entire folders or multiple files at once.
- **Dark Mode:** Respects system preferences and includes a manual toggle.

## Tech Stack

- **Frontend:** HTML5, CSS3 (Modern variables & Glassmorphism), JavaScript (Vanilla)
- **Libraries:** PDF.js for PDF extraction, Mammoth.js for Word documents.
- **Build Tool:** Parcel

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deployment

This project is optimized for deployment on **Vercel**. Simply connect your GitHub repository and it will auto-detect the Parcel build settings.
